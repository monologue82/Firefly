## 终于上线了！

经过几个月的开发和迭代，**智能教室管理系统（Smart Classroom）** 终于正式上线了！

👉 **访问地址：[https://app.monologue82.xyz](https://app.monologue82.xyz)**

如果你是一名班主任、任课老师，或者只是想让班级事务更有序，这个系统就是为你准备的。

## 它能做什么？

### 积分管理 — 告别手工记账

还在用小本子记学生的加分扣分？现在不需要了。

- 给每个学生建立积分档案
- 一键加分/扣分，还能写明原因
- 自动生成排行榜，谁进步了一目了然
- 支持批量操作，全班同时加减分也不在话下

### 作业管理 — 布置、追踪、归档

- 在线发布作业，设置科目和截止日期
- 自动标记"即将到期"、"今天到期"、"已逾期"
- 历史作业随时翻查，期末复习有据可依

### 卫生值日 — 排班不再头疼

- 自动生成值日表，公平分配任务
- 记录每天完成情况，谁做的一清二楚
- 历史记录可追溯，期末评优有数据支撑

### 随机抽号 — 课堂互动好帮手

- 点击按钮，随机抽取一名学生
- 带动画效果，学生看得清清楚楚
- 抽取范围可自定义，想抽谁就抽谁

### 数据大屏 — 班级状况一目了然

- 实时显示班级积分、作业、值日等关键数据
- 卡片式布局，打开就知道今天该关注什么

## 为什么选它？

### 完全免费，零服务器成本

系统托管在 **Cloudflare** 的免费套餐上，不需要你买服务器、买域名（当然你可以绑定自己的域名）。Cloudflare 的全球 CDN 让访问速度飞快，无论你在学校还是家里，打开即用。

### 多班级独立管理

每个班级有独立的账户和数据，互不干扰。你是班主任？注册一个班级就行。你教多个班？每个班一个账户，切换方便。

### 数据安全

- 登录使用 JWT 认证，密码加密存储
- 每个班级只能看到自己的数据
- 支持数据导出和备份，你的数据你做主

### 界面美观，操作简单

- 现代化 UI 设计，支持亮色/暗色主题
- 响应式布局，电脑、平板、手机都能用
- 操作逻辑简单，不需要培训就能上手

## 技术栈

| 层面 | 技术 |
|------|------|
| 前端 | React 18 + HeroUI + Tailwind CSS |
| 构建 | Vite 5 |
| 后端 | Cloudflare Pages Functions（无服务器） |
| 数据库 | Cloudflare D1（边缘 SQLite） |
| 部署 | Cloudflare Pages（全球 CDN） |
